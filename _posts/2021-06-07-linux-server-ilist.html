记录一下linux系统中常用的socket API

常用API列表

函数名 函数功能描述 其他
socket 创建一个套接字  
bind 将一个套接字绑定到<ip, port>的二元组  
listen 将一个socket变为侦听状态  
connect 建立tcp连接 通常在客户端
accept 接收一个连接 通常在服务端
send 通过一个socket发送数据  
recv 通过一个socket接收数据  
select 判断 一组 socket上的读写和异常事件  
gethostbyname 通过域名获取机器地址  
close 关闭一个套接字，并且回收其对应的资源 Windows下是closesocket
shutdown 关闭一个socket或收发通道  
setsockopt 设置一个套接字选项  
getsockopt 获取一个套接字选项
